Return

Value returned by your function


							function setup() {
							 createCanvas(400,400);
							 rectMode(CENTER;)
							}
							function draw() {
							 background(235);
							 let h = calculateHeight(mouseX, mouseY);
							 fill(50);
							 square(100, 100, h); // value of h received from calculateHeight Function
							}
							function calculateHeight(a, b){
							 let height = a - b; //datatype Number used for subtraction and calculating
							 return height;
							}

On a canvas with dimensions 600 x 400, make a sketch that fulfils these conditions

  • Make a function that creates a pattern(s) using loop(s)
  • Colors on this sketch change based on time, a feature that can be switched on/off
  • This feature(mentioned above) is controlled with a button, that you'll make
  • Pressing 'x' on the keyboard switches the button OFF, which turns off the color-changing. Pressing 's' turns ON the button and feature
  • The only text, if required, that can exist on the sketch, can be on the 'button'. Character limit is 8 at the max
  • The intricacies of the pattern should respond to the mouse movement on your sketch
